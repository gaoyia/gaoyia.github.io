# /api/nav/getNavArr

一次返回多个导航，例如经常有顶部导航和底部导航同时使用就可以使用这个接口提高接口请求效率

# 参数

参数名 参数类型 说明 是否可选 默认值
ids Array id的数组 否 -
type Int 返回的菜单类型 是 true(1)

# 接口返回示例

{
    "code": 200,
    "data": [
        {
            "id": 1,
            "zh_cn": "顶部导航",
            "nav_list": [
                ...{}
            ]
        },
        {
            "id": 2,
            "zh_cn": "底部导航",
            "nav_list": [
                ...{}
            ]
        }
    ],
    "msg": "请求成功"
}

# 请求错误参考

{
    "code": 404,
    "msg": "导航不存在"
}
